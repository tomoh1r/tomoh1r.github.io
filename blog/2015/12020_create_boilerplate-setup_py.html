boilerplate-setup.py を作ってみた¶

2015/12/02

いくつか package を作っていて setup.py のひな形が欲しいなって思っていた。

ちょうど ansible-vault の p-r を対応する際に tox/travis 対応などをしたので、まとめとして boilerplate-setup.py を作成した。

spam_ham_egg/spam-ham-egg の箇所を書き換えたり、名前とか書き換えればつかえると思う。

ご活用ください。

追記¶

cookiecutter に対応しました。

$ pip install cookiecutter
$ cookiecutter https://github.com/jptomo/boilerplate-setup.py

で使えます。
